### Ample - Game engine
2D, physics-friendly

### Project manager
You can create ample directories anywhere, using ample script.

### Install (Linux)

`$ ./ci/travis/install-linux.sh` \ - installs dependencies
`$ mkdir build && cd build` \
`$ cmake .. && cmake --build`


### Create your project (in developing)
`$ cd buile/gui` \
`$ ./ample-gui`


### Run (demo)
`$ cd build/demo` \
`$ ./demo`
